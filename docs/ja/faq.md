# よくある質問（FAQ）

## セットアップ

### Q: Bot Token はどこで取得しますか？

[Discord Developer Portal](https://discord.com/developers/applications) で Application を作成し、Bot セクションで Token をリセット・コピーします。詳しくは [インストール・初期設定](getting-started.md) を参照してください。

### Q: 「許可ユーザーIDが設定されていません」と表示されます

`antiCrow.allowedUserIds` に自分の Discord ユーザー ID を設定してください。Discord の「詳細設定」で「開発者モード」を有効にし、自分のユーザー名を右クリック → 「ユーザーIDをコピー」で取得できます。

### Q: Bot が起動しません

以下を確認してください:

1. Bot Token が正しく設定されているか（コマンドパレット → 「AntiCrow: Set Bot Token」）
2. Antigravity が起動しているか
3. `antiCrow.autoStart` が `true` になっているか

## 基本操作

### Q: メッセージを送信しても反応がありません

以下を確認してください:

1. ステータスバーに「Active」と表示されているか
2. `antiCrow.allowedUserIds` に自分の ID が含まれているか
3. Bot が正しいサーバーに招待されているか
4. Antigravity の接続状態（`/status` コマンドで確認）

### Q: タスクを途中で止めたい

`/stop` コマンドを使ってください。実行中のジョブがキャンセルされます。

### Q: 前のタスクが終わる前に次のメッセージを送った場合は？

自動的にキューに追加されます。前のタスクが完了すると、順番に処理されます。キューの状態は `/queue` コマンドで確認できます。

### Q: AI のモデルを変更したい

`/model` コマンドでモデル一覧を表示し、ボタンをクリックして切り替えてください。

## 連続オートモード

### Q: 連続オートモードが途中で止まってしまいます

以下の原因が考えられます:

- セーフティガードが発動した（危険なコマンドの検出）→ 承認/スキップ/停止を選択
- `--confirm` が `semi` や `manual` に設定されている → `auto` に変更
- `--steps` の上限に達した → `--steps` の値を増やす
- `--duration` のタイムアウト → `--duration` の値を増やす

### Q: 連続オートモードが使えません

連続オートモードは **Pro プラン限定** の機能です。`/pro` コマンドからアップグレードしてください。

## エージェントチームモード

### Q: エージェントチームモードを使うにはどうすればよいですか？

1. Pro プランにアップグレード
2. `/team` → 「🟢 チーム ON」でエージェントチームモードを有効化
3. 通常通りメッセージを投稿（AI が自動でタスクを分割）

### Q: 「リポジトリにコミットがありません」と表示されます

エージェントチームモードは Git worktree を使用するため、最低1つのコミットが必要です:

```bash
git init && git add -A && git commit -m "initial commit"
```

## ライセンス

### Q: Free プランの制限は？

1日あたりのタスク実行回数と1週間あたりの実行回数に制限があります。連続オートモードとエージェントチームモードは使用できません。

### Q: ライセンスキーを忘れました

Lemonsqueezy の購入確認メールにライセンスキーが記載されています。

### Q: Pro プランの料金は？

♾️ **Lifetime ($50)** の買い切りプランのみです。一度購入すれば永久に全機能を無制限に使用できます。

## トラブルシューティング

### Q: 「Antigravity との接続が初期化されていません」と表示されます

Antigravity が起動していることを確認してください。起動していても表示される場合は、AntiCrow を再起動してみてください（コマンドパレット → 「AntiCrow: Stop」→ 「AntiCrow: Start」）。

### Q: レスポンスがタイムアウトします

`antiCrow.responseTimeoutMs` を確認してください。0（無制限）に設定するか、値を大きくすると改善する場合があります。`antiCrow.maxRetries` でリトライを有効にすることもできます。

### Q: SOUL.md が長すぎてモーダルで編集できません

Discord モーダルは4000文字が上限です。テキストエディタで直接 SOUL.md ファイルを編集してください。
