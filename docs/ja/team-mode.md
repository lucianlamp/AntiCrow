# チームモード

> 👥 複数の AI サブエージェントが並行してタスクを処理（Pro 限定）

## 概要

チームモードは、大きなタスクを複数のサブエージェント（Antigravity の別ウインドウ）に分割し、並行して処理させる機能です。例えば、3つの独立したファイル修正を3つのサブエージェントが同時に行うことで、処理時間を大幅に短縮します。

## 使い方

### チームモードの有効化

1. `/team` コマンドを実行
2. 「🟢 チーム ON」ボタンをクリック

### チームモードの無効化

1. `/team` コマンドを実行
2. 「🔴 チーム OFF」ボタンをクリック

### タスクの実行

チームモードが有効な状態で通常のメッセージを投稿すると、AI が以下を判断します:

- タスクが**分割可能**な場合 → 自動的にサブエージェントに分配
- タスクが**分割不要**な場合 → メインエージェントで単独実行

### 分割の判断基準

AI は以下の基準でタスクを分割します:

**分割する（チームモード使用）:**

- 3ファイル以上にまたがる変更
- 新機能の実装＋テスト＋デプロイが必要な作業
- 独立した複数の問題を同時に修正する作業
- 調査・実装・検証が別々に並行可能な作業

**分割しない（メインエージェント単独）:**

- 単一ファイルの修正・設定変更
- 情報の確認・質問への回答
- 簡単なバグ修正（1-2ファイル以内）
- 型チェック・テスト・ビルドのみの実行

## チーム管理パネル

`/team` コマンドで以下の情報を確認・操作できます:

| 項目 | 説明 |
|------|------|
| チームモード | 有効 / 無効 |
| 稼働中サブエージェント | 現在活動中のエージェント数 |
| タイムアウト | サブエージェントの最大実行時間 |
| 監視間隔 | 進捗確認の間隔 |
| 自動スポーン | サブエージェントの自動起動 |

## ヘルプモード 🤝

チームモードには**ヘルプモード**が内蔵されています。先に自分のタスクを完了したサブエージェントが、まだ完了していない他のタスクを自動的に手伝います。

### ヘルプの優先順位

1. テスト作成
2. ドキュメント更新
3. コードレビュー
4. 未着手の関連作業

### 制約

- 他のエージェントが作業中のファイルは**絶対に上書きしない**
- 各サブエージェントは独立した Git worktree で作業

## 注意事項

- チームモードは **Pro プラン限定** の機能です
- リポジトリに最低1つのコミットが必要です
- 各サブエージェントは独立した Antigravity ウインドウで動作します
- 実行完了後、変更は自動的にメインブランチに統合されます
