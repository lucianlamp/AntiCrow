# 料金プランと制限

AntiCrow には **Free プラン** と **Pro プラン** の2種類があります。

## プラン比較

| 機能 | Free | Pro |
|------|------|-----|
| プラン生成（AI計画立案） | 無制限 | 無制限 |
| タスク実行（日次） | 10回/日 | 無制限 |
| タスク実行（週次） | 50回/週 | 無制限 |
| 連続オートモード | ✅ | ✅ |
| エージェントチームモード | ✅ | ✅ |
| autoApprove（自動承認） | ❌ | ✅ |

## Free プランの制限

### タスク実行回数

- **日次制限**: 1日あたり10回まで
- **週次制限**: 1週間あたり50回まで

### プラン生成は無制限

AntiCrow にメッセージを送ると、まずAIが「何をするか」の計画（プラン）を生成します。
このプラン生成は **Free プランでも無制限** です。

制限がかかるのは、プランを承認して **実際にタスクを実行する** ときだけです。

### 残りタスク数の表示

タスク実行後、残りの実行可能回数が自動的に表示されます：

```
📊 残り: 8/10 回（本日）| 45/50 回（今週）
```

### 制限に達した場合

制限に達すると、タスク実行がブロックされ、Proプランへのアップグレードボタンが表示されます。

## Pro プラン

- タスク実行回数の制限なし
- autoApprove（自動承認）が利用可能
- 買い切り型（月額なし）

詳細は [Pro プラン](pro-plan.md) を参照してください。
