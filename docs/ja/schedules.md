# スケジュール実行

> ⏰ タスクを定期的に自動実行する

## 概要

スケジュール機能を使うと、タスクを cron 式で定期実行できます。毎日のレポート作成、定期的なテスト実行、バックアップなど、繰り返しのタスクを自動化できます。

## スケジュールの作成

### 方法 1: メッセージから自動判定

Discord にメッセージを送信するとき、AI が「即時実行」と「定期実行」のどちらかを自動判定します。

```
毎日9時にデイリーレポートを作成して
```

→ AI が定期実行と判断し、cron スケジュールを提案します。

### 方法 2: `/schedules` コマンドから作成

1. `/schedules` コマンドを実行
2. 「➕ 新規作成」ボタンをクリック
3. モーダルで以下を入力:
   - **名前**: スケジュール名（例: `daily-report`）
   - **実行日時**: 自然言語で指定（例: `毎日 9時`）
   - **実行プロンプト**: 実行する内容

### 日時の指定方法

自然言語で指定できます:

| 入力例 | cron 式 |
|--------|---------|
| `毎日 9時` | `0 9 * * *` |
| `毎週月曜 14:30` | `30 14 * * 1` |
| `月曜と水曜の 14:30` | `30 14 * * 1,3` |
| `毎月1日 10:00` | `0 10 1 * *` |
| `毎週金曜 18:00` | `0 18 * * 5` |

## スケジュールの管理

`/schedules` コマンドで管理パネルを表示します。

### 操作ボタン

| ボタン | 動作 |
|--------|------|
| ⏸️ / ▶️ | スケジュールの一時停止 / 再開 |
| ✏️ | スケジュールの編集（名前・日時・プロンプト） |
| 🗑️ | スケジュールの削除 |
| ▶️ 即時実行 | スケジュールを今すぐ1回実行 |
| ➕ 新規作成 | 新しいスケジュールを追加 |

## 結果の通知

定期実行の結果は、対象ワークスペースの Discord チャンネルに自動通知されます。

## 注意事項

- スケジュールは Antigravity が起動している間のみ有効です
- Antigravity を再起動しても、登録済みスケジュールは保持されます
- タイムゾーンは設定に従います（デフォルト: JST / `Etc/GMT-9`）
