# 連続オートモード

> 🤖 AI が自律的にタスクを連続実行するモード

## 概要

連続オートモード（`/auto`）は、AI に複雑なタスクを任せて自律的に連続処理させる機能です。1つのステップが完了するたびに AI が次のアクションを判断し、自動で実行を続けます。

## 使い方

### 基本

```
/auto LPをリニューアルしてください
```

### オプション付き

```
/auto --steps 5 --confirm semi --duration 30 バグを全部修正して
```

## オプション

| オプション | 値 | デフォルト | 説明 |
|-----------|-----|-----------|------|
| `--steps` | 1〜20 | 10 | 最大ステップ数 |
| `--duration` | 5〜120 | 60 | 最大実行時間（分） |
| `--confirm` | `auto` / `semi` / `manual` | `auto` | 確認モード |
| `--select` | `auto-delegate` / `first` / `ai-select` | `auto-delegate` | 選択方式 |

## 確認モード

### `auto`（自動実行）

⚡ 全ステップを確認なしで自動実行します。最も効率的ですが、意図しない操作が行われるリスクがあります（セーフティガードが保護します）。

### `semi`（偶数ステップで確認）

🔄 偶数ステップ（2, 4, 6...）で一時停止し、ユーザーに続行・停止の確認を求めます。自動と手動のバランスが取れたモードです。

### `manual`（毎ステップ確認）

✋ すべてのステップ完了後に確認を求めます。最も安全ですが、操作が多くなります。

## 選択方式

### `auto-delegate`（自律判断）

🤖 AI にタスクの進め方を完全に委任します。提案が表示されている場合は AI が最適な提案を選び、なければ自律で次のアクションを決定します。

### `first`（最初の提案）

1️⃣ 前のステップで提案された選択肢のうち、最初のものを自動選択します。

### `ai-select`（AI に選択させる）

🧠 提案一覧を AI に見せ、最も適切なものを AI が選択して実行します。

## セーフティガード 🛡️

連続オートモード実行中、以下の危険なアクションが検知されると**自動的に一時停止**します:

### ブロック（実行前に確認必須）

| カテゴリ | 検知パターン例 |
|----------|--------------|
| ファイルシステム破壊 | `rm -rf`, `format`, ディスク操作 |
| Git 破壊操作 | `git push --force`, `git reset --hard` |
| データベース破壊 | `DROP TABLE`, `TRUNCATE TABLE` |
| 秘密鍵アクセス | `private_key`, `secret_key`, `mnemonic` |
| 暗号資産操作 | `transfer`, `swap`, `withdraw all` |
| 外部送信 | `curl + secret`, `post + mnemonic` |
| 環境変数漏洩 | `.env` ファイルの出力 |
| コード注入 | `eval`, `exec()`, `Function()` |

### セーフティ発動時のボタン

| ボタン | 動作 |
|--------|------|
| ✅ 承認 | この操作を許可してループを再開 |
| ⏭️ スキップ | このステップをスキップして次に進む |
| 🛑 停止 | 連続オートモードを完全停止 |

## ステップの流れ

```
🚀 連続オートモード開始
   ↓
📝 ステップ 1 実行
   ↓
✅ ステップ 1 完了 → 提案リスト生成
   ↓
🤖 AI が次の提案を選択 (選択方式に基づく)
   ↓
📝 ステップ 2 実行
   ↓
   ... (maxSteps or maxDuration まで繰り返し)
   ↓
📊 連続オートモード完了
```

## 完了通知

連続オートモード完了時に表示される情報:

- ✅ 完了ステップ数 / 最大ステップ数
- ⏱️ 合計時間
- 🛡️ セーフティ発動回数

### 提案ボタン

連続オートモード完了時、最後のステップで AI が提案を生成していた場合、完了サマリーと一緒に**提案ボタン**が表示されます。ボタンをクリックすると、そのアクションを即座に実行できます。

| ボタン | 説明 |
|--------|------|
| 💡🔧🚀 提案 | AI が提案した次のアクション（最大3つ） |
| 🤖 エージェントに任せる | AI の判断で最適なアクションを実行 |
| 🔄 連続オートモードで実行 | 提案を基に連続オートモードを再実行 |

提案がない場合はボタンは表示されません。

## 停止方法

連続オートモードを途中で停止するには:

1. `/stop` コマンドを実行
2. セーフティ発動時に「🛑 停止」ボタンをクリック
3. 確認モードで「🛑 停止」ボタンをクリック

## 設定の永続化

`/auto` で指定した設定は**チャンネルごとに保存**されます。次回同じチャンネルで `/auto` を使うときは、前回の設定がデフォルトとして適用されます。

## 注意事項

- 実行中は `autoApprove`（確認ダイアログの自動承認）が一時的に有効になります
- 連続オートモード終了後、`autoApprove` は元の設定に戻ります
- セーフティガードは常に有効です（無効化できません）
