## Pro トライアル

初回インストール時に Pro トライアル期間が提供されます。トライアル中は全機能を無制限に使用できます。

## ライセンスの購入

### Discord から購入

1. `/pro` コマンドを実行
2. 「♾️ Lifetime ($50)」ボタンをクリック
3. 購入ページで決済を完了
4. ライセンスキーをメールで受け取る

### ライセンスキーの入力

1. `/pro` コマンドを実行
2. 「🔑 キー入力」ボタンをクリック
3. モーダルに Lemonsqueezy のライセンスキーを貼り付け
4. 認証成功メッセージを確認

### コマンドパレットから入力

1. Antigravity でコマンドパレット（`Ctrl+Shift+P`）を開く
2. 「AntiCrow: License Login」を選択
3. ライセンスキーを入力

## ライセンス情報の確認

- `/pro` → 「📋 ライセンス情報」ボタン
- コマンドパレット → 「AntiCrow: License Info」

表示される情報:

- 現在のプラン名
- ライセンスキーの状態

## ライセンスのリセット

ライセンスキーを削除して Free プランに戻す場合:

- コマンドパレット → 「AntiCrow: License Info」から操作

## オフライン猶予

ネットワーク接続がない場合でも、一定期間はライセンス認証済みとして動作し続けます（オフライン猶予期間）。
