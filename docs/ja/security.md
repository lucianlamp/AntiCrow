# 🛡️ セキュリティポリシー

AntiCrow はユーザーのセキュリティとプライバシーを最優先に設計されています。

---

## アーキテクチャ概要

AntiCrow はすべての処理をユーザーのローカル環境内で完結させます。

```
📱 Discord（スマホ/PC）
    ↓ メッセージ送信
🐦‍⬛ AntiCrow 拡張機能（ローカル PC）
    ↓ タスク変換
🤖 Antigravity AI（ローカル PC）
    ↓ 結果返却
🐦‍⬛ AntiCrow 拡張機能（ローカル PC）
    ↓ 結果送信
📱 Discord（スマホ/PC）
```

**通信は Discord API とのやりとりのみです。** それ以外の外部サーバーへのデータ送信は一切行いません。

---

## データの取り扱い

### 保存されるデータ

| データ | 保存場所 | 保護方法 |
|---|---|---|
| Discord Bot Token | Antigravity SecretStorage | OS レベルの暗号化 |
| ライセンスキー | Antigravity SecretStorage | OS レベルの暗号化 |
| 実行計画（プラン） | ローカル JSON ファイル | ユーザー権限で保護 |
| タスク実行カウンター | Antigravity globalState | ローカル永続化（日次/週次リセット） |
| カスタム AI 指示 | `~/.anticrow/ANTICROW.md` | ユーザーが完全に管理 |

### 送信されないデータ

- ❌ テレメトリ・使用統計データ
- ❌ Analytics やトラッキング情報
- ❌ ユーザーのファイルシステム構造
- ❌ 外部の第三者 API へのデータ送信
- ❌ ライセンスキーの外部送信（検証は暗号化通信のみ）
- ❌ タスク実行回数・制限情報の外部送信

### 添付ファイル

Discord 経由で送られたファイルはローカルに一時ダウンロードされ、AI の処理後に自動削除されます。外部サーバーにアップロードされることはありません。

---

## アクセス制御

- **`allowedUserIds`** — 許可された Discord ユーザーのみが Bot を操作できます。空の場合は**全ユーザーを拒否**します（ホワイトリスト方式）。
- **Bot Token** — Token を知っている管理者のみが Bot を制御できます。

### ⚠️ Discord アカウントが乗っ取られた場合

Discord アカウントが不正アクセスを受けた場合、以下の手順で AntiCrow への不正操作を即座に遮断できます：

1. Antigravity の設定から `antiCrow.allowedUserIds` を開く
2. 乗っ取られたアカウントのユーザー ID を **削除**（または全 ID を空にして全拒否）
3. 設定を保存 → **即座に反映**（Bot の再起動不要）
4. 必要に応じて Discord Developer Portal で **Bot Token をリセット**

> 💡 `allowedUserIds` はホワイトリスト方式のため、ID を削除するだけで即座にアクセスを遮断できます。

## ライセンスとプランのセキュリティ

AntiCrow は Free プランと Pro プランを提供しています。ライセンスに関わるデータは以下のように安全に管理されます。

### ライセンスキーの保管

- ライセンスキーは Antigravity の **SecretStorage** に暗号化保存されます
- 設定ファイルに平文で記録されることはありません
- ライセンス検証は Lemonsqueezy API への HTTPS リクエストのみで行われます

### 実行制限の管理

- Free プランの実行制限（日10回/週50回）はローカルに保存・管理されます
- 実行回数や制限情報が外部に送信されることはありません
- Pro トライアル情報もローカルに保存され、外部に送信されません

## ユーザーがコントロールできる項目

AntiCrow はユーザーが完全にコントロールできるように設計されています：

| 設定 | 説明 |
|---|---|
| `allowedUserIds` | 誰が Bot を操作できるかを制御 |
| `~/.anticrow/ANTICROW.md` | AI に送られるカスタム指示を自分で編集・確認 |
| `workspacePaths` | AI がアクセスするフォルダを制御 |
| `responseTimeoutMs` | タイムアウト時間を制御 |

> 💡 `ANTICROW.md` は平文の Markdown ファイルです。AI に送信されるカスタム指示の内容をいつでも確認・編集できます。
