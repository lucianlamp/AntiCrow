# テンプレート機能

> 📋 よく使うプロンプトを保存して、ワンクリックで実行

## 概要

テンプレート機能を使うと、よく使うプロンプトを名前をつけて保存し、ボタン1つで実行できます。変数を使って実行時にパラメータを動的に変更することも可能です。

## 使い方

### テンプレート一覧を表示

```
/templates
```

保存済みテンプレートがボタン付きリストで表示されます。

### 新規作成

1. `/templates` コマンドを実行
2. 「➕ 新規作成」ボタンをクリック
3. モーダルでテンプレート名とプロンプトを入力
4. 保存

### テンプレートの実行

1. `/templates` コマンドを実行
2. 実行したいテンプレートの「▶ 実行」ボタンをクリック
3. カスタム引数がある場合はモーダルで値を入力
4. プレビュー表示後、実行

### テンプレートの削除

1. `/templates` コマンドを実行
2. 削除したいテンプレートの「🗑️ 削除する」ボタンをクリック
3. 確認後に削除

## 変数

テンプレートのプロンプトに以下の変数を使用できます:

### 組み込み変数

| 変数 | 説明 | 例 |
|------|------|-----|
| `{{date}}` | 今日の日付 | `2026-03-08` |
| `{{time}}` | 現在の時刻 | `21:00` |
| `{{datetime}}` | 日付と時刻 | `2026-03-08 21:00` |
| `{{year}}` | 年 | `2026` |
| `{{month}}` | 月 | `03` |
| `{{day}}` | 日 | `08` |

### 環境変数

```
{{env:VARIABLE_NAME}}
```

OS の環境変数を展開します。

### カスタム引数

```
{{引数名}}
```

任意の名前でカスタム引数を定義できます。実行時にモーダルで値を入力できます（最大5個）。

## テンプレート例

### デイリーレポート

**名前:** `daily-report`

**プロンプト:**

```
{{date}} のデイリーレポートを作成してください。
今日のコミットログを確認し、変更内容をまとめてください。
```

### デプロイ

**名前:** `deploy`

**プロンプト:**

```
{{branch}} ブランチを本番環境にデプロイしてください。
テストが通ることを確認してからデプロイすること。
```

→ 実行時に `branch` の値をモーダルで入力

### コードレビュー

**名前:** `code-review`

**プロンプト:**

```
最新のコミットをレビューしてください。
バグ、セキュリティリスク、パフォーマンスの問題を確認し、改善点を報告してください。
```
