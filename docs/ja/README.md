# 🐦‍⬛ AntiCrow ユーザーガイド

> Discord から Antigravity を自動操作するブリッジ拡張機能

## AntiCrow とは？

AntiCrow は、Discord のメッセージを通じて Antigravity（AI コーディングアシスタント）を自動操作できる拡張機能です。自然言語で依頼を送信するだけで、即時実行・定期実行が可能です。結果は Discord にリアルタイムで通知されます。

スマホからでも PC からでも、Discord さえあれば AI にコーディングを依頼できます。

## 主な機能

- 📱 **Discord からの AI 操作** — メッセージを送るだけで Antigravity が動く
- ⚡ **即時実行 & 定期実行** — 今すぐ実行も cron スケジュールも対応
- 🤖 **自律モード** — AI が自律的にタスクを連続実行（Pro 限定）
- 👥 **チームモード** — 複数のサブエージェントが並行作業（Pro 限定）
- 📋 **テンプレート** — よく使うプロンプトを保存して1クリック実行
- 🛡️ **セーフティガード** — 危険なコマンドを自動検知・ブロック
- 🔐 **セキュリティ** — 許可ユーザーのみ操作可能
