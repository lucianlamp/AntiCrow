# Pricing & Limits

AntiCrow offers two plans: **Free** and **Pro**.

## Plan Comparison

| Feature | Free | Pro |
|---------|------|-----|
| Plan generation (AI planning) | Unlimited | Unlimited |
| Task execution (daily) | 10/day | Unlimited |
| Task execution (weekly) | 50/week | Unlimited |
| Continuous Auto Mode | ✅ | ✅ |
| Agent agent team mode | ✅ | ✅ |
| Auto Approve | ❌ | ✅ |

## Free Plan Limits

### Task Execution Limits

- **Daily limit**: Up to 10 tasks per day
- **Weekly limit**: Up to 50 tasks per week

### Plan Generation is Unlimited

When you send a message to AntiCrow, the AI first generates a plan for what to do.
This plan generation is **unlimited even on the Free plan**.

Limits only apply when you approve a plan and **actually execute a task**.

### Remaining Task Count Display

After each task execution, your remaining task count is automatically displayed:

```
📊 Remaining: 8/10 (today) | 45/50 (this week)
```

### When You Reach the Limit

When you reach the limit, task execution is blocked and an upgrade button to the Pro plan is displayed.

## Pro Plan

- No task execution limits
- Auto Approve available
- One-time purchase (no monthly fees)

For details, see [Pro Plan](pro-plan.md).
