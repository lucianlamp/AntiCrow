## Purchasing a License

### From Discord

1. Execute the `/pro` command
2. Click "♾️ Lifetime ($69.99)"
3. Complete payment on the purchase page
4. Receive your license key via email

### Entering Your License Key

1. Execute the `/pro` command
2. Click the "🔑 Enter Key" button
3. Paste your Lemonsqueezy license key in the modal
4. Confirm the success message

### Via Command Palette

1. Open the Antigravity Command Palette (`Ctrl+Shift+P`)
2. Select "AntiCrow: License Login"
3. Enter your license key

## View License Info

- `/pro` → "📋 License Info" button
- Command Palette → "AntiCrow: License Info"

Displays:

- Current plan name
- License key status

## Reset License

To remove your license key and return to the Free plan:

- Command Palette → "AntiCrow: License Info" → operate from there

## Offline Grace Period

Even without network connectivity, your license continues to work for a grace period (offline grace period).
