# 🔒 プライバシーポリシー

最終更新: 2026年2月18日

AntiCrow（以下「本拡張機能」）は、ユーザーのプライバシーを尊重し、データの収集を最小限に抑えるよう設計されています。

---

## 基本方針

- 本拡張機能は **外部サーバーにデータを送信しません**
- すべての処理は **ユーザーのローカル PC 上で完結** します
- テレメトリ、Analytics、使用統計の収集は **一切行いません**

---

## 収集・使用するデータ

### Discord メッセージ

- **用途:** ユーザーの指示を AI タスクに変換するためにのみ使用
- **保存:** 実行計画（プラン）としてローカルの JSON ファイルに保存
- **送信先:** ローカルの Antigravity AI のみ（外部サーバーには送信しません）

### Discord Bot Token

- **用途:** Discord API との接続認証
- **保存:** Antigravity の SecretStorage に暗号化して保存
- **アクセス:** 本拡張機能のみがアクセス。平文での保存や外部への送信はしません

### 添付ファイル

- **用途:** AI による分析・処理
- **保存:** ローカルの一時ディレクトリにダウンロード
- **削除:** 処理完了後に自動削除
- **送信先:** ローカルの Antigravity AI のみ

### カスタム AI 指示（`ANTICROW.md`）

- **用途:** AI への追加指示として全プロンプトに適用
- **保存:** ユーザーのホームディレクトリ（`~/.anticrow/ANTICROW.md`）
- **管理:** ユーザーが完全にコントロール。内容はいつでも確認・編集可能

---

## 収集しないデータ

本拡張機能は以下のデータを **収集・送信しません:**

- 使用統計やテレメトリ
- クラッシュレポート
- ユーザーの個人情報
- ファイルシステムの構造やファイル一覧
- ブラウザの閲覧履歴
- 他の拡張機能の情報

---

## 第三者サービスとの通信

本拡張機能が通信する外部サービスは **Discord API のみ** です。

| 通信先 | 用途 | データ内容 |
|---|---|---|
| Discord API | Bot のメッセージ送受信 | メッセージテキスト、リアクション、添付ファイル |

上記以外の第三者サービスへの通信は一切行いません。

---

## データの削除

本拡張機能に関連するすべてのデータを削除するには：

1. **拡張機能のアンインストール:** Antigravity の拡張機能パネルから AntiCrow を削除
2. **ローカルデータの削除:** `~/.anticrow/` ディレクトリを削除
3. **Bot Token の無効化:** [Discord Developer Portal](https://discord.com/developers/applications) で Token をリセット
